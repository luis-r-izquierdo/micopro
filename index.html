How to install the model

To use MiCoPro, you will have to install NetLogo 6 (free and open source) and download this .zip file. The .zip file contains the model itself (micopro.nlogo) and also this website (which contains a description of the model). To run the model, you just have to click on the file micopro.nlogo.

License


MiCoPro (Mixing in Contagion Processes) is a model designed to analyze the consequences of mixing two groups that have different predispositions to adopt a certain trait (or infection).
